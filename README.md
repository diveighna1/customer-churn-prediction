# Customer Churn Prediction Analysis

## Overview
This project analyzes customer churn behavior using machine learning techniques to identify factors that contribute to customer retention and cancellation. The goal was to build predictive models capable of identifying customers at higher risk of churn and generate insights to support data-driven business decisions.

## Objectives
- Analyze customer behavioral data
- Identify key factors influencing churn
- Build and compare machine learning models
- Evaluate predictive model performance

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Machine Learning Models
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## Project Workflow
1. Data cleaning and preprocessing
2. Exploratory data analysis (EDA)
3. Model training and testing
4. Model evaluation and comparison

## Key Findings
- Customers with shorter tenure showed a higher likelihood of churn
- Payment behavior and customer engagement patterns influenced retention
- Random Forest achieved the strongest overall model performance
